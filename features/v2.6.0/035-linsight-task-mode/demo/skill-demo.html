C3 契约速览（display_name + 多形态上传扩展）

端点（/api/v1/linsight 前缀）

GET/skill管理列表，PageData[SkillBrief]，get_tenant_admin_user
GET/skill/selectable终端选择器，仅已启用，get_login_user
GET/skill/{name}详情（frontmatter + 正文 + bundle 文件树 files[]）
GET/skill/{name}/file?path=读取 bundle 附属文件内容（只读 · 新增）
POST/skillmultipart（.md / .zip / .skill / 文件夹打包）或 form 新建，默认启用
PUT/skill/{name}编辑：form 仅改 SKILL.md；multipart 整包替换（保持同一技能 ID）
PATCH/skill/{name}/status{enabled:bool} 启停
DEL/skill/{name}删除（前端二次确认）

built-in 不在 /skill 命名空间：对其任何操作一律 404（不泄露存在性）。压缩包/文件夹：根目录须含 SKILL.md，附属文件随技能目录整体落盘，整包 ≤10MB。

frontmatter 规范

name: 必填 ≤64 ^[a-z0-9]+(-[a-z0-9]+)*$，须=目录名 description: 必填 ≤1024 allowed-tools / license: 可选 metadata: 可选 dict[str,str] display-name: 展示名称（缺省回退 name） sop-id: 迁移溯源（仅迁移产物） 单 .md 或 zip 整包 ≤10MB；zip/.skill/文件夹根目录须含 SKILL.md
