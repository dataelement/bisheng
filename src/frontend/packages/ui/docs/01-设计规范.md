## 1. 圆角 Radius → 已迁出

2026-07-23 起圆角规范独立成文，见 [基础-阴影与圆角规范.mdx](基础-阴影与圆角规范.mdx)：控件档 4/6/8px 随尺寸档走、容器档 12/16/24/32px 随容器大小走、胶囊用 full；**弹窗定 16px**（8 vs 16 之争就此了结，A 套 8px 列为待迁移旧值）；嵌套圆角 = 外层 − 内边距；禁任意值，存量归并映射见其附录 A。本节旧内容作废，以新文档为准。

## 2. 间距 Spacing

### 已定

| 场景 | 值 | 出处 |
|---|---|---|
| 同组相邻按钮（紧凑/工具栏） | 8px | 按钮规范 §4 |
| 弹窗 footer 按钮间距 | 12px（`gap-3`） | 按钮规范 §4（对齐本文） |
| 弹窗 footer 按钮 min-width | 60px；移动端等宽平铺 | 按钮规范 §3.1 / §5.5 |
| 二次确认弹窗内边距（C 套，已落地） | 20px（`p-5`） | Modal 文档改动记录 2026-07-02 |

### 待定

| 场景 | 现状 | 待拍板 |
|---|---|---|
| Modal 弹窗内边距 | A 套 `p-5`(20) vs 手拼 AlertDialog `p-6`(24) 并存 | 20 vs 24 二选一（Modal 文档待决策清单）；C 套先例是 20 |
| 表单项垂直间距 | 未统计 | 建议 16px（`space-y-4`），随 Input 组件期定 |

## 3. 阴影 / 投影 Shadow → 已迁出

2026-07-23 起阴影规范独立成文，见 [基础-阴影与圆角规范.mdx](基础-阴影与圆角规范.mdx)：只留两档——浮层投影 `0 2px 16px -2px rgba(0,23,66,0.10)`（下拉/气泡）与模态投影 `0 0 16px 0 rgba(3,7,117,0.05)`（弹窗/抽屉），两档取值即下拉与弹窗基准组件已落地值，一律搭配 1px 边框、静态区块不加投影、禁散写 `shadow-*`。本节旧的「三种现状 + 两档建议」作废，以新文档为准。

## 5. 层级 z-index

### 现状（三档并存，不一致）

| 层 | z-index |
|---|---|
| B 套弹窗 | `z-50` |
| A 套弹窗 | `z-[100]` |
| AlertDialog（C 套底座） | `z-[110]` |

### 待定稿

- 建议统一一套层级阶梯：遮罩 + 弹窗一档，Drawer / Sheet / Popover / Tooltip 各归其位，写成 token 表。
- 统一时需**先盘 Drawer / Sheet / Popover 与弹窗的叠放关系**（Modal 文档 §二点六 已提示），避免改完出现弹窗被抽屉盖住类回归。
